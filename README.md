# photo-finder


 ## About
 
 Photo Finder is an application where you can search for beautiful photos by any keyword you choose.
 
 ## Setup & Install
 
 - `git clone https://github.com/KylieStanley/photo-finder.git`
 - `cd photo-finder`
 - `npm install`
 - `npm run serve`
 

 ## Tech Stack
  - Vue.js
